Display Control+
================

A professional Windows utility for OLED and LCD display protection, overlay, and control.

INSTALLATION
------------
- Portable (single-file): download DisplayControlPlus.exe from the GitHub Releases page and run it (supports --background for the service path).
- Installer: download and run DisplayControlPlusSetup.exe, then choose desktop shortcut and tray auto-start options as desired. After install, Display Control+ will launch and appear in your system tray.

USAGE
-----
- Right-click the tray icon for options: Show Overlay, Settings, Help/About, Quit.
- Use the GUI to select monitors, overlay mode, and images.
- Click 'Apply' to save settings. Overlays/screensavers will activate as configured.
